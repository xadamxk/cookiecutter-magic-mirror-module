# MagicMirror Module Cookiecutter

[Cookiecutter](https://github.com/cookiecutter/cookiecutter) recipe to easily create [MagicMirror](https://github.com/MichMich/MagicMirror) [Modules](https://github.com/MichMich/MagicMirror/wiki/3rd-Party-Modules).

## Features

- Translations support
- Nunjucks template integration
- External request/automatic refresh examples
- Custom configuration parameters
- Message passing using Notifications (Global Module <=> Node Helper)

## Requirements

- [Python3](https://www.python.org/downloads/)
- [Cookiecutter](https://cookiecutter.readthedocs.io/en/stable/installation.html#install-cookiecutter)
  > Note: Windows users may need to add `Python/Scripts` to their PATH ([read more](https://cookiecutter.readthedocs.io/en/stable/installation.html#windows))

## Usage

```
cookiecutter https://github.com/xadamxk/cookiecutter-magic-mirror-module.git
```
